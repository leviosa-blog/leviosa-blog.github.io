(二) 分支语句

1.单分支语句 (if)

任何能够产生 True 或 False 的语句都可以作为条件，当条件为真时，执行语句块中的内容。

2.二分支语句(if else)

当条件为 True 时，执行语句块 if下的命令；当条件为 False 时，执行语句块 else下的命令。

3.多分支语句(if elif else)

依次判断条件并执行对应 的语句。若没有条件成立，执行 else 下面的语句。
